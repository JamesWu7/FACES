FACES

Facial Analysis for Classification of Etiology in Scoliosis is a deep learning research prototype for hierarchical screening and etiological stratification of pediatric scoliosis from frontal facial photographs.

Open Hugging Face demo View GitHub repository

Hierarchical Workflow

1. Screening

Binary model output for healthy control versus scoliosis-related support.

2. Etiology

Superclass ranking across AIS, CMS and syndromic scoliosis.

3. SS Subtyping

Ranked support across 11 syndromic scoliosis subtype categories.

Use Boundary

FACES is intended for research demonstration and clinical triage support only. Model outputs are not standalone diagnoses and should be interpreted together with radiographs, MRI, genetic testing, history, physical examination and specialist assessment.

Deployment

  • Hugging Face Spaces: runs the real Gradio/PyTorch inference app.
  • GitHub Pages: hosts this static information page only; it does not process uploaded images or run model inference.
  • Local or intranet deployment: recommended for clinical demonstrations involving identifiable facial photographs.

Privacy

Facial photographs are identifiable medical data. Public demos should use consented demonstration images only. Raw facial images, patient identifiers and individual-level clinical data should not be committed to GitHub.
